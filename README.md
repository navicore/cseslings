# CSES Problem Set - Rust Practice

Working through the [CSES Problem Set](https://cses.fi/problemset/) in Rust.

## Getting Started

1. Go to [cses.fi/problemset](https://cses.fi/problemset/) for problem descriptions and test input
2. Navigate to a problem: `cd 01-introductory-problems/01-weird-algorithm`
3. Edit `src/main.rs` with your solution
4. Test locally: `cargo run < input.txt`
5. Submit on the CSES website

## Regenerating the Workspace

To start fresh or add new problems:

```bash
# Remove all problem directories
rm -rf [0-9][0-9]-*/

# Regenerate from script
./setup_cses.sh
```
